#The Basics#

##Extended window##

###getScrollLeft()###
Gets the left-scroll offset of the window.

###getScrollTop()###
Gets the top-scroll offset of the window.

###getWidth()###
Gets the width of the window.

###getHeight()###
Gets the height of the window.
